// 物品掉落公告配置文件
// 
// rAthena自带的 conf/battle/drops.conf 中的 rare_drop_announce 选项是
// 根据物品掉率来判定是否需要进行公告的。
// 
// 然而有时候，我们希望只要是特定的道具，不管是什么掉率，一旦掉落就公告。
// 如果有这种需求的话，可以使用本配置文件来解决问题。
//
// 格式为:
// 物品编号,公告的触发时机
//
// 公告的触发时机:
//	&1: 打死魔物，魔物将道具掉落到地上时公告
//	&2: 打死魔物，掉落MVP奖励到玩家的背包时公告
//	&4: 使用“偷窃”技能，从魔物身上偷到物品时公告
//
// 例如:
// 909,1   // 当打死魔物，魔物将杰勒比结晶掉落在地上时公告
// 713,3   // 当打死魔物，魔物将空瓶掉落在地上或当做MVP奖励直接放到玩家背包时公告
//
// 注意:
// 您会发现“开箱子时，获得了物品并不会公告”，这是因为在 rAthena 的官方设计中，
// 用来开箱子的 getgroupitem 指令拥有它自己的一套公告体系，具体请见 doc/item_group.txt
